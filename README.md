# Himno - Repositorio de Imágenes

Repositorio centralizado para todas las imágenes y assets visuales del proyecto Himno.

## 📁 Estructura de Carpetas

```
himno-images/
├── logos/                 # Logos del proyecto (PNG, SVG)
├── backgrounds/           # Fondos e imágenes de fondo
├── assets/               # Otros assets visuales (iconos, banners, etc.)
└── icons/               # Iconos individuales
```

## 📝 Descripción de Carpetas

### `logos/`
Contiene los logos principales del proyecto Himno en diferentes formatos y tamaños:
- Logo principal
- Logo alternativos
- Variantes (color, blanco y negro, etc.)

**Formatos recomendados:** PNG, SVG

### `backgrounds/`
Fondos e imágenes de fondo para la aplicación:
- Fondos para hero sections
- Texturas
- Patrones

**Formatos recomendados:** PNG, JPG, WebP

### `assets/`
Otros elementos visuales del proyecto:
- Banners
- Ilustraciones
- Gráficos promocionales

### `icons/`
Iconos individuales utilizados en la interfaz:
- Iconos de navegación
- Iconos de acciones
- Iconos temáticos

**Formatos recomendados:** SVG, PNG

## 🚀 Cómo Usar

### Referenciar imágenes desde el repositorio principal

```markdown
![Descripción](https://raw.githubusercontent.com/dmardonesv/himno-images/main/logos/himno-logo.png)
```

### En HTML
```html
<img src="https://raw.githubusercontent.com/dmardonesv/himno-images/main/logos/himno-logo.png" alt="Himno Logo">
```

### En CSS (fondo)
```css
background-image: url('https://raw.githubusercontent.com/dmardonesv/himno-images/main/backgrounds/bg-main.png');
```

## 📐 Estándares de Imágenes

- **Logos:** Mínimo 512x512 px (preferentemente SVG)
- **Backgrounds:** 1920x1080 px o superior
- **Icons:** 64x64 px, 128x128 px (múltiplos de 32)
- **Optimización:** Comprimir imágenes para web

## 📦 Formatos Soportados

- ✅ PNG
- ✅ SVG
- ✅ JPG/JPEG
- ✅ WebP
- ✅ GIF
